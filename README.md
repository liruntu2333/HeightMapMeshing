# HeightMapMeshing
A D3D11 visualizer of Height Map Meshing based on [GitHub - fogleman/hmm: Heightmap meshing utility.](https://github.com/fogleman/hmm)
glm installed by vcpkg
```bash
vcpkg install glm:x64-windows
```
